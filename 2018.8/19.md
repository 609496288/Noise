- MySQL位置查询：

  ```
  安装目录：select @@basedir;
  数据文件的存放目录：select @@datadir;
  ```

- Mac更新locate数据库：

  ```
  updatedb位置在/usr/libexec/locate.updatedb
  使用sudo扩大权限更新数据库，但还是无法搜索到
  sudo /usr/libexec/locate.updatedb
  ```
